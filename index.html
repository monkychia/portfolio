Stunning Examples

YumYumNav

YumYumNav a single page web site that allows users to enter a city or zip code or restaurant's name to where they are travelling to. In return, we will display a list of restaurants within the city or zip code or restaurant's name entered by the user. Within the list of restaurants, user can click on any one of them to list the details of the specific restaurant.
